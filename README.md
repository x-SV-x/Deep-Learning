# Contents : 
1. **Single Layer Perceptron** - Implementation of a single layer perceptron from scratch.
2. **Multi Layer Perceptron** - **Binary Classification** into the two classes - cancer/not-cancer using ANNs (Multi layer perceptrons) trained on a ovarian tumor dataset. I also implment different regularization techniques : $Dropout$, $Early$ $stopping$, $L1$ $and$ $L2$ $regularization$, $Batch$ $normalization$; and evaluate performances for each.
3. **Convolutional Neural Networks** - A CNN model implemented on a MRI dataset for Alzheimers.
